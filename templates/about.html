What is dinner wizard?

On this page you will learn all you need to know to get started and save time on your dinner and grocery shoppings plans

1. Getting started

Step one to getting started with Dinner Wizard is to create a plan, so let's do that.

Click on the "Create New Plan" button in your emtpy active plan list.

2. Create a Plan

Here you can give your plan a fitting name, For this example we call it "My first plan".

As this is likely your first plan, go ahead and set it to active. The default is upcoming, more on that later.

Go ahead and click create. As you can see, we will add recipes to this plan in the next step

If you forget to do this, don't worry. We can make it active later on.

3. Create recipe

After creating your plan, you are taken to the details page for the plan. Here you can add recipes.

Notice that the recipes box is empty as you havn't added any recipes yet. Let's do that now.

There are already many recipes created by other users that you can use by clicking the "Add Recipe" button, but more on that later.

For now, let's use the "Create New Recipe" button.

4. Create your first recipe

Clicking the "Create New Recipe" button will take you to the recipe creation.

Write a fitting name for your recipe, something that will make it obvious for others what it is.

In the description you write how to create the recipe. If you wnat to, you can start the text with a presentation of your recipe.

Next step is to select categories for your recipe, there are many to choose from, so pick the ones that fit best. You can select one, many or none. It is all up to you.

As with recipes when you created your plan, we will add ingredients to the recipe in the next step.

Note: You do not need to write a list of ingredients in the description, we will add then in the next step.

5. The recipe details page

Just as when you created a plan, when you click create on a recipe, you are taken to the details page.

Again you will find an empty box for ingredients, so let's go ahead and add some. Click in the "Add Ingredients" button.

6. Add Ingredients

Here you will find a list of ingredients that you can add. You can scroll through the list to find it, or use the filter button above the list to filter by name.

If you can't find the ingredient you need, you can create a new one. Below the list you will find two fields, "name" and "unit". Add information to them and click "Create Ingredient" to add it. Note: This will not add it to your recipe, just add it to the list of possible ingredients

When you find the ingredient you want, click on the "+" to add it. All you need to do is add a quantity and click "Add"

After click add you will be returned to your recipe. Keep adding ingredients until you have all you need. When you are satisfied click the "Done" button.

You will now be taken automatially to the plans overview page (read more about it below). Notice that your new recipe is automatically added to your plan.

You can create recipes witohut doing so from a plan, with the link in the top navigation or the button on the plans page. These will not be added to a plan, however.

7. Add Recipe

If you want to add an existing recipe to your plan, click the "Add Recipe" button from your plan details page.

Here you will find a list of recipes, simply select the ones you want and click add. You can select multiple recipes at once using the ctrl/cmd-key.

In order to help you find the ones you want more quickly, there is a filer function. Read more about it in the advanced usage section.
